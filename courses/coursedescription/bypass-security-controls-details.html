What You Will Master

  • Evade firewalls and network ACLs using tunneling and packet fragmentation.
  • Bypass modern Endpoint Detection & Response (EDR) using API unhooking and direct system calls.
  • Circumvent application whitelisting by abusing trusted "Living off the Land" binaries (LOLBins).
  • Bypass Web Application Firewalls (WAFs) using payload encoding and tamper scripts.
  • Defeat Multi-Factor Authentication (MFA) with techniques like session cookie theft and push fatigue.
  • Exploit authorization flaws like Insecure Direct Object References (IDOR) and Path Traversal.
  • Find and exploit common IAM misconfigurations in cloud environments like AWS.
  • Perform high-speed, offline password cracking of dumped hashes using tools like Hashcat.
