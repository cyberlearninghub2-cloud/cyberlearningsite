Android Application Security

A masterclass on finding and exploiting vulnerabilities in Android apps, from reverse engineering and static analysis to advanced runtime manipulation.

100% Free Course

What You Will Master

  • Reverse engineer Android APKs using `apktool` and decompile DEX bytecode to Java.
  • Perform dynamic instrumentation with Frida to hook methods and manipulate app logic in real time.
  • Bypass runtime protections like root detection and SSL certificate pinning to enable analysis.
  • Conduct a security review of the Android Manifest to find insecure, exported components.
  • Identify hard-coded secrets, weak cryptography, and insecure data storage in source code.
  • Test for common API vulnerabilities like BOLA/IDOR by intercepting and manipulating traffic.
  • Exploit vulnerabilities in native code libraries (e.g., buffer overflows) using Ghidra or IDA Pro.
  • Conduct a comprehensive, end-to-end security assessment based on the OWASP MASVS.
