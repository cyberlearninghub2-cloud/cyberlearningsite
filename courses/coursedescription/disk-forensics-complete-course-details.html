Disk Forensics: A Complete Course

A comprehensive 20-lesson course on the principles and practices of digital disk forensics, from evidence acquisition to advanced file system and artifact analysis.

100% Free Course

What You Will Master

  • Follow the forensic investigation process from evidence handling to reporting.
  • Create and verify forensically sound disk images using tools like `dd` and `FTK Imager`.
  • Perform in-depth analysis of the NTFS file system, including the MFT and USN Journal.
  • Analyze Linux file systems like ext4 and understand concepts like inodes.
  • Carve and recover deleted files and data from unallocated disk space.
  • Conduct timeline analysis to reconstruct user activity and system events.
  • Analyze Windows artifacts like the Registry, LNK files, and Prefetch files.
  • Write a professional forensic report detailing your findings and conclusions.
