Sandboxing & Detection Engineering

A comprehensive course on designing and building automated sandboxing systems. Learn to turn threat analysis into high-fidelity detection rules in this defense-focused curriculum.

100% Free Course

What You Will Master

  • Design and build a complete, isolated analysis lab environment with VMs and network simulators.
  • Analyze process, file system, registry, and network behavior using tools like Process Monitor and Wireshark.
  • Use memory forensics with Volatility to detect injected code and fileless threats.
  • Identify common malware evasion techniques like packing, anti-VM checks, and staged loading.
  • Write high-fidelity, behavior-based detection rules using formats like Sigma and Falco.
  • Apply machine learning concepts for automated malware classification based on behavioral features.
  • Enrich sandbox findings with external threat intelligence from sources like VirusTotal and MITRE ATT&CK.
  • Automate the reporting and sharing of threat intelligence in formats like STIX and MISP.
