Network Forensics

A comprehensive course on network traffic analysis for incident response and threat hunting. Learn to capture, decode, and reconstruct network activity to uncover the full story of a cyberattack.

100% Free Course

What You Will Master

  • Master packet analysis with Wireshark and Tshark to dissect TCP/IP, DNS, and HTTP/S traffic.
  • Detect and analyze malware C2 communications, beaconing, and data exfiltration channels like DNS tunneling.
  • Perform wireless forensics to identify rogue access points and investigate attacks against 802.11 networks.
  • Analyze logs from firewalls, IDS/IPS, proxies, and other security devices to reconstruct intrusions.
  • Conduct network forensics in cloud environments like AWS and Azure, focusing on VPC flow logs and API analysis.
  • Investigate threats in specialized environments like Industrial Control Systems (ICS/SCADA).
  • Use threat intelligence and automation (SOAR) to drive proactive threat hunting campaigns.
  • Synthesize and correlate network, host, and memory evidence to build a master incident timeline.
